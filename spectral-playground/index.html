How It Works

This visualization shows how different optimizers transform gradient singular values:

  • SGD: f(x) = x - No transformation, gradients are used as-is
  • Muon: Multiple iterations of g(x) = 3/2 * x - 1/2 * x³ to push singular values toward 1
  • Reverso: Approximates x/(x² + 1e-6) to handle large singular values

The color gradient helps you see how values from the raw histogram map to the transformed histogram.
